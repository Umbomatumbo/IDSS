The structure consits of five main folders:
1. Data
2. Documentation
3. Presentation
4. Models
5. Source


The Data folder contains the file data.csv with the original data used for this work.
The Documentation folder contains the pdf report
The Presentation folder contains the pdf with the slides for the presentation.
The Models folder includes the model and a notebook for viewint the results.
The Source folder contains three subfolder
1. API_and_Models = The python code for the API and the different models used 
2. Data_Analysis_training = Contains the notebook used for data analysis and training
3. WebAPP = Contains the web application. The importan files are in the path: src/sections/predictor

The web app is publicly available at: http://idss.eastus.cloudapp.azure.com
